Proyectos

Aquí encuentras la aplicación de proyectos innovadores

Llama2

Aplicación de Llama2, un avanzado modelo de lenguaje implementado como chatbot, incluye la capacidad de generar resúmenes precisos a partir de documentos extensos o compuestos por múltiples páginas.

Analizador de datos

Esta aplicación proporciona un entorno eficiente para la manipulación de datos. Puedes realizar tareas avanzadas como la eliminación de duplicados, el manejo de valores nulos, la generación de visualizaciones y la realización de predicciones basadas en la variable seleccionada.

Link

Predicción Vivienda

Aplicación de modelos de Deep Learning y Machine Learning como herramientas que permiten a los usuarios predecir el valor estimado de mercado de una vivienda, ya sea para venta o arriendo, ubicación y barrio.

Link

Solubilidad Química

Aplicación de machine learning para estimar la solubilidad acuosa de Delaney, propiedades de importancia y la visualización de su distribución de cargas en 2D y 3D de las moléculas. Además, la implementación de un balanceador de ecuaciones químicas. 

Link

Análisis de Mercado

El objetivo de este proyecto es analizar el mercado de acciones y llevar a cabo predicciones para los valores futuros de las acciones utilizando Deep Learning. Para ello, se cuenta con un conjunto de datos de datos históricos de precios de acciones, que se emplea para entrenar un modelo de redes neuronales.

Link

Extractor de Texto

Aplicación que utiliza tecnología OCR avanzada para extraer texto de imágenes con precisión y rapidez. Ideal para digitalizar documentos, convertir imágenes a texto editable y mejorar la eficiencia en la gestión de información visual.
